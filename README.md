# ALAN_Sleep_ReefFish
# Overview
This repository serves as the digital archive for the data and analysis code associated with the manuscript:
"Chronic artificial light pollution disrupts sleep and neuronal genomic stability in wild reef fish"

## Data & Code Availability
This repository is currently being prepared for public release. 

All raw data files and custom analysis scripts will be fully uploaded and made publicly available in this repository immediately upon aceptence publication of the manuscript.

## Repository Structure (Planned)
Upon release, this repository will be organized as follows:
* `code/`: Custom R and Python scripts for behavioral tracking and statistical analysis.
* `data/`: Raw datasets for field observations and laboratory experiments (CSV format).
* `figures/`: Code to reproduce the main figures of the manuscript.
